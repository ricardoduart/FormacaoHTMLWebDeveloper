História da Web

Primeiros Computadores

Voltar
Mark I: Primeiro computador eletromecânico construído por Howard Aiken em parceria com a IBM em 1944, pioneiro no processamento automático de cálculos.
Colossus: Primeiro computador eletrônico programável, desenvolvido na Inglaterra durante a Segunda Guerra Mundial para decifrar códigos alemães.
ENIAC: Primeiro computador eletrônico de grande escala, concluído em 1945, projetado para cálculos complexos e científicos, representando um avanço na computação digital.

Internet - Como Surgiu

Voltar
Internet surgiu a partir da década de 1960 como um projeto de pesquisa militar nos EUA, evoluindo de redes de computadores interligadas para compartilhar informações. O desenvolvimento do protocolo TCP/IP nos anos 70 permitiu a comunicação entre diferentes sistemas, culminando na criação da World Wide Web por Tim Berners-Lee em 1989, tornando a Internet acessível e navegável para o público em geral.

Grandes Nomes

Voltar

Joseph Carl Robnett Licklider
Teorizou sobre uma rede onde todos poderiam se comunicar através dela. Desde então muita coisa aconteceu…
Vinton Cerf e Robert Kahn
Cerf e Robert Kahn são fundamentais para o surgimento da Internet devido à sua coautoria no desenvolvimento do protocolo TCP/IP (Transmission Control Protocol/Internet Protocol), uma estrutura essencial que permitiu a comunicação eficiente e interconexão de redes de computadores. O TCP/IP estabeleceu padrões para transmitir dados em pacotes pela rede, tornando possível a troca de informações entre diferentes sistemas e pavimentando o caminho para a expansão global da Internet. Seu trabalho inovador foi crucial para o funcionamento e a arquitetura da rede que conhecemos hoje.
Tim Berners-Lee
Em 1989, Tim Berners-Lee viu a oportunidade de unir hipertexto com TCP/IP e criar a World Wide Web (WWW). Primeiro navegador chamado WorldWideWeb

O que são Clients

Voltar

Cliente: É o dispositivo ou software que inicia uma solicitação de serviço ou conteúdo. Isso pode ser um navegador da web, um aplicativo móvel ou qualquer outro programa que envie pedidos a um servidor para obter dados ou funcionalidades.

Navegadores

Voltar Navegadores são programas de empresas para abrir/arquivos executar, seguindo padrões W3C, com interpretações variáveis. Também chamados de "browsers", começaram com o Mosaic, evoluíram com Netscape e agora há muitas opções disponíveis gratuitamente.

Aplicações Web

Voltar
Aplições Web são soluções criadas que possuem a internet como meio de comunicação entre Client x Server não sendo necessário a sua instalação.

Dispositivos Móveis

Voltar
Dispositivos móveis são eletrônicos portáteis, como smartphones e tablets, que permitem comunicação, acesso à internet, aplicativos e funções diversas em movimento.

O que são Servers

Voltar
Servidores são computadores especializados que fornecem serviços, recursos ou dados a outros dispositivos, chamados clientes, por meio de redes, como a Internet. Eles armazenam, gerenciam e distribuem informações, como sites, aplicativos, arquivos e e-mails, atendendo às solicitações dos clientes de maneira eficiente e confiável.

Hospedagem de Sites

Voltar
Hospedagem de sites é o serviço de alugar espaço em servidores para armazenar e disponibilizar um site na internet, garantindo seu acesso e funcionamento global.

Linguagens de Programação

Voltar
Linguagem de Programação é uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas (software).

Server-Side

Voltar
Linguagens de programação Server-Side são utilizadas para desenvolver a lógica e as funcionalidades dos servidores de um sistema web. Elas processam solicitações vindas de clientes, como navegadores, e geram respostas dinâmicas, como páginas web personalizadas ou dados manipulados. Exemplos incluem: PHP, Python, Ruby, Java e Node.js.

Client-Side

Voltar
Linguagens de programação Client-Side são usadas para desenvolver a lógica e interatividade diretamente nos navegadores e dispositivos dos usuários. Elas controlam a aparência e o comportamento das páginas da web após serem baixadas do servidor. Exemplos incluem: JavaScript, HTML e CSS.

Estrutura Básica do HTML

Voltar

A Estrutura Básica do HTML é um esqueleto fundamental para criar páginas da web. Consiste em elementos como <!DOCTYPE>, <html>, <head>, e <body>. O <!DOCTYPE> especifica a versão do HTML, enquanto <html> envolve todo o conteúdo. O <head> contém informações sobre a página, como título e metadados, e o <body> abriga o conteúdo visível, como texto, imagens e elementos interativos.

Links

Voltar

Em HTML, links são elementos que permitem que os usuários naveguem entre diferentes páginas da web. Eles são criados usando a tag <a> (âncora) e são definidos através do atributo href, que contém o endereço da página de destino. Ao clicar em um link, os usuários podem ser redirecionados para outras páginas, recursos ou locais na mesma página. Links podem ser usados para conectar páginas dentro do mesmo site, abrir URLs externas, enviar e-mails e muito mais, tornando-os uma parte fundamental da navegação na web.
